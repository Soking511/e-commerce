E-Commerce Website
This project is a full-stack E-Commerce Website built using the MEAN Stack (MongoDB, Express.js, Angular, Node.js). It allows users to browse, search for products, add them to the cart, and complete the purchase process. The website also includes an admin dashboard for managing products, categories, users, orders, and other essential e-commerce features.
